### 자주 묻는 질문

**Q. 차단이 동작하지 않아요.**
설정 → 일반 → VPN 및 기기 관리 → Screen Time 권한이 켜져 있는지 확인해주세요. 권한이 없으면 차단·스케줄·엄격 모드 모두 작동하지 않습니다.

**Q. 엄격 모드를 풀고 싶어요.**
엄격 모드는 설정한 시간이 끝나기 전엔 어떤 방법으로도 풀 수 없도록 설계되어 있어요. 비밀번호 · 앱 삭제 · 재설치 모두 무효입니다. 시간이 지날 때까지 기다려주세요.

**Q. 친구 랭킹이 안 보여요.**
설정 앱 → Apple ID → iCloud → iCloud Drive 가 켜져 있는지 확인해주세요. iCloud 로그인 + iCloud Drive 활성화가 필요합니다.

**Q. 데이터를 삭제하고 싶어요.**
- 차단 설정 / 점수 등 모든 로컬 데이터: 앱을 삭제하면 함께 사라집니다.
- 랭킹 기록 (CloudKit): 앱 안 관리 패널에서 "내 랭킹 기록 삭제" 또는 위 이메일로 요청하세요.

### FAQ

**Q. Blocking doesn't work.**
Please check Settings → General → VPN & Device Management → Screen Time permission is granted. Without it, blocking / scheduling / strict mode will not function.

**Q. I want to disable strict mode.**
Strict mode is designed to be unstoppable until the configured time ends. Passwords, app deletion, or reinstallation will not bypass it. Please wait until the timer expires.

**Q. Friend leaderboard isn't showing.**
Please check Settings app → Apple ID → iCloud → iCloud Drive is enabled. iCloud sign-in + iCloud Drive must both be active.

**Q. I want to delete my data.**
- Blocking settings / focus scores / all local data: deleting the app removes everything.
- Ranking record (CloudKit): use the in-app admin panel "Delete my ranking record", or email us at the address above.
